# Prompt Attributes
- **Contextual Attributes**: background information or context provided to the model.
  - 
- **Stylistic Attributes**: define the tone, style, and form of the output.
  - Formal or informal
  - Detailed or non-detailed
- **Format Attributes**: refers to the structure and format of the output.
  - Length
  - Specific elements to include
  - Response format
